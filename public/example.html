ChatFlow Integration Demo

This page demonstrates how to integrate the ChatFlow widget into any website using a single script tag.

Installation

Copy and paste the following snippet into your HTML's <body>:

<script src="https://ais-dev-u5p5kdtg2ii36nzbftwvyx-702146563588.europe-west2.run.app/chatbot.js"></script>
<script>
  window.onload = function() {
    Chatbot.init({
      apiKey: "DEMO_API_KEY",
      botName: "Demo Bot",
      welcomeMessage: "Welcome! I'm the ChatFlow demo bot. How can I help?",
      primaryColor: "#4CAF50"
    });
  };
</script>

The chatbot should appear in the bottom-right corner of this page.

Check the console for logs. If the script is loaded correctly, you'll see "Powered by ChatFlow".
